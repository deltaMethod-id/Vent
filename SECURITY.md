# Security Policy

## 🔐 About Security

Security is an important part of Vent Note.
We appreciate researchers and users who help us find and report security issues.

## 📢 Reporting a Security Vulnerability

If you discover a security vulnerability in Vent Note, please report it responsibly.

Please include:

- A clear description of the issue
- Steps to reproduce the vulnerability
- Expected behavior
- Actual behavior
- Screenshots or additional evidence (if available)

## 🚫 Please Do Not

- Do not access other users' data
- Do not perform attacks that can harm the service
- Do not publicly disclose a vulnerability before it is fixed

## 🛠️ Supported Versions

| Version | Supported |
|---------|-----------|
| Latest version | ✅ |
| Older versions | ❌ |

## ⏱️ Response Time

We will review security reports as soon as possible and provide updates when available.

Thank you for helping make Vent Note safer.
