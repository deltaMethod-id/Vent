# 🌙 Vent

**Vent** is a simple web application that allows users to share their thoughts and feelings in a safe and minimal environment.

## ✨ Features

- 🔐 Vent Account (Sign Up & Login)
- 💾 Local Storage Database
- 📝 Create Posts
- ✏ Edit Your Posts
- 🗑 Delete Your Posts
- 👤 Change Username
- 🔑 Change Password
- 🚪 Logout
- 🗑 Delete Account
- 📱 Responsive Design
- ⚡ Fast & Lightweight
